## Description

The "How To" dropdown menu in the main navigation provides access to tutorial/help videos for store managers. This is a global navigation element available on all pages.

| How to Update Kit Status and Reorder Kit Items | `/dashboard/video/kit_status_and_reorder` |

---

## UX Observations

1. **Limited Options:** Only 2 help videos available - may need more comprehensive help documentation
2. **Video Format:** Links go to video pages rather than text documentation
3. **Positioning:** Menu appears left-aligned under the "How To" link
4. **No Search:** No way to search help topics
5. **No Categories:** With only 2 items, categories aren't needed, but may be needed as content grows
6. **Accessibility:** Could benefit from keyboard navigation hints
