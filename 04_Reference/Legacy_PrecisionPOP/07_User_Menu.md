## Description

The user account dropdown menu appears when clicking on the user name in the header navigation. It provides access to account settings and logout functionality.

---

## Menu Items

| Menu Item | Link | Purpose |
|-----------|------|---------|
| Account Info | `/users/manage` | Manage user profile/settings |
| Privacy Policy | `/pages/privacy` | View privacy policy |
| Log Out | `/users/logout` | End session and log out |

---

## UX Observations

1. **User Identification:** Shows full role and store info "Good2Go Store Manager Store #119"
2. **Minimal Options:** Only 3 menu items - very simple
3. **No Profile Picture:** User icon is generic, no avatar/photo option visible
4. **No Notifications:** No notification settings or preferences
5. **No Password Change:** Account Info may include this, but not explicitly listed
6. **Clean Design:** White dropdown with clear text, good contrast
7. **Missing:** 
   - Help/Support contact
   - Language/locale settings
   - Theme preferences
   - Switch store (if user manages multiple)
