# Missing Screen: Detailed Campaign Analytics

## Location
Regional Dashboard -> Campaigns (Specific Item View)

## Description
While a general campaign list exists, the deep-dive analytics for a specific campaign in a specific region are missing.

## Expected Content
- **Regional Drill-down**: Heatmap or chart showing compliance by store within the region.
- **Compliance Funnel**: Progression of stores through (Received -> Installed -> Photographed -> Verified).
- **Issue Breakdown**: Common reasons for failure (Missing parts, Wrong location, Poor photo quality).
- **Communication Hub**: Interface to message all store managers in the region about a specific campaign delay.
