4. COLOR CODING
--------------------------------------------------------------------------------

  Orange  : Offline status, warnings, pending items
  Green   : Success, completed syncs, online
  Red     : Errors, failed uploads
  Blue    : Informational, capabilities
  Gray    : Disabled, waiting states

--------------------------------------------------------------------------------
5. USER FLOWS
--------------------------------------------------------------------------------

  Going Offline:
  1. Network disconnects
  2. WiFi icon changes to wifi-off (orange)
  3. Orange banner appears
  4. Features needing connection grey out

  Working Offline:
  1. Capture photo -> Queued locally
  2. Complete survey -> Saved to IndexedDB
  3. Badge shows pending count
  4. Banner shows last sync time

  Coming Back Online:
  1. Network reconnects
  2. Auto-sync starts
  3. Items upload one by one
  4. Queue badge updates
  5. Sync history adds success entries
  6. Banner disappears when queue empty

  Checking Sync Status:
  1. Tap Profile tab
  2. Tap "Sync Status" (see pending badge)
  3. View queued items
  4. Retry failed uploads
  5. See sync history
  6. Learn offline capabilities
