## 11 Capability Pillars

1. **Brand-First SaaS Model** - Multi-PSP, brand-led evolution
2. **Digital Asset Management (DAM)** - Full asset lifecycle management
3. **AI Intelligence** - Data analytics and image manipulation
4. **Online Designer** - In-platform design tools
5. **Online Proofing** - Approval workflows
6. **Workflow Automation** - Integration and automation
7. **MIS/ERP** - Production management for PSPs
8. **Native Mobile** - iOS/Android applications
9. **White-Label** - Reseller and branding programs
10. **Training Academy** - LMS and certifications
11. **Marketplace** - Designer/installer network

---

## Phase Evolution

```
v1 (Current)  → PSP-led campaign orchestration
v2            → Brand configuration, basic DAM, mobile apps
v3            → Multi-PSP routing, AI features, certifications
v4            → Full marketplace, enterprise features
v4+           → Network effects, industry standard
```

---

## Key Principle

**v1 must ship first.** These documents inform future direction but do not justify v1 scope expansion. Each phase must prove value before the next begins.
