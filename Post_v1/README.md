## Key Principle

**v1 is PSP-led.** Future versions may become brand-led or network-led, but v1 must ship first.

Nothing in this folder should be used to justify v1 scope expansion. These documents exist for:
- Strategic planning conversations
- Architecture decisions that preserve optionality
- Post-v1 roadmap discussions

---

## Evolution Path

```
v1 (Current)        → PSP-led campaign orchestration
v1.x                → Additional PSP/brand onboarding
v2                  → Brand-led configuration options
v3                  → Multi-PSP network / SaaS marketplace
```

Each phase builds on the previous. v1 must prove the core loop before evolution begins.
