# Onboarding Process

## Overview
This document outlines the comprehensive onboarding process for all PopSystem user types, ensuring new customers achieve first value quickly and establish successful long-term usage patterns.

## Onboarding Philosophy

### Core Principles

**Time to Value:**
- Get users to their first "aha moment" within 7 days
- Reduce friction at every step
- Provide clear, achievable milestones
- Celebrate early wins

**Personalization:**
- Tailor onboarding to user role and use case
- Adjust pace based on customer sophistication
- Provide relevant examples and templates
- Adapt to industry-specific needs

**Education over Training:**
- Focus on "why" not just "how"
- Build confidence through progressive disclosure
- Encourage exploration and experimentation
- Provide ongoing learning resources

**Success Measurement:**
- Clear success criteria for each phase
- Data-driven progress tracking
- Early intervention for at-risk customers
- Continuous optimization of onboarding flow

## PSP Onboarding (30-Day Journey)

### Pre-Onboarding (Days -7 to 0)

**Sales-to-CS Handoff:**
- Deal close notification to CS team
- Review sales notes and customer goals
- Assign Customer Success Manager
- Review contract and commitments
- Prepare customized onboarding plan

**Welcome Communication:**
- Immediate welcome email from CSM (within 2 hours of close)
- Kickoff call scheduling (within 48 hours)
- Access credentials and login instructions
- Pre-kickoff homework: Gather brand assets, user list, campaign examples
- Quick-start video (5 min overview)

**Account Provisioning:**
- Account creation with company branding
- Initial user roles configured
- Integration credentials prepared
- Sample data populated (optional)
- Onboarding checklist activated

### Day 1: Welcome and Account Setup

**Kickoff Call (60 minutes):**

**Agenda:**
```
1. Introductions (10 min)
   - CSM introduction and role
   - Customer team introductions
   - Identify champion and key stakeholders

2. Goal Setting (15 min)
   - Review customer objectives
   - Define success criteria
   - Establish KPIs to track
   - Timeline alignment

3. Platform Overview (20 min)
   - Navigation and key modules
   - Role-based access demonstration
   - Account settings and customization
   - Quick tour of first campaign creation

4. Onboarding Plan Review (10 min)
   - 30-day timeline and milestones
   - Training session schedule
   - Resource overview
   - Communication preferences

5. Q&A and Next Steps (5 min)
   - Address immediate questions
   - Confirm next meeting
   - Action items
```

**Deliverables:**
- Kickoff call recording and notes
- Customized onboarding checklist
- Training schedule
- Primary contact list
- Resource library access

**Post-Call Actions:**
- Send meeting summary and action items
- Activate account with full permissions
- Schedule all training sessions
- Send admin guide and quick-start resources

### Week 1: Foundation (Days 2-7)

**Objectives:**
- Complete account configuration
- Admin and champion proficiency
- First campaign created (not necessarily executed)
- Team members invited

**Day 2-3: Account Configuration**

**Self-Guided Tasks:**
- Company profile completion
- Branding customization (logo, colors)
- User management setup
- Notification preferences
- Billing information verification

**CSM Support:**
- Configuration review call (30 min, optional)
- Template library access
- Best practice documentation
- Integration preparation

**Day 4: Admin Training Session (90 minutes)**

**Training Agenda:**
```
1. User Management (25 min)
   - Creating user accounts
   - Role-based permissions
   - Team organization
   - SSO setup (Enterprise)

2. Account Settings (20 min)
   - Company information
   - Branding and customization
   - Notification rules
   - API access (if needed)

3. Campaign Basics (35 min)
   - Campaign structure overview
   - Creating your first campaign
   - Store network setup
   - Campaign templates
   - Scheduling and assignment

4. Q&A and Practice (10 min)
   - Hands-on exercise
   - Questions and troubleshooting
```

**Deliverables:**
- Training recording
- Admin quick reference guide
- User invitation template
- Campaign template library

**Day 5-7: First Campaign Setup**

**Guided Activities:**
- Campaign creation walkthrough
- Store network configuration
- Product/SKU setup
- Task and workflow definition
- Mobile app preview

**Milestone Checkpoint:**
- Day 7 check-in call (30 min)
- Review onboarding progress (target: 40% complete)
- Address any blockers
- Adjust timeline if needed
- Celebrate quick wins

**Success Criteria for Week 1:**
- Account fully configured
- Admin trained and comfortable
- 3+ team members invited
- First campaign created
- Store network added (at least partial)

### Week 2: Enablement (Days 8-14)

**Objectives:**
- Staff training completed
- Campaign workflow configured
- Mobile app onboarded for field teams
- Integration setup (if applicable)
- Test campaign execution

**Day 8-9: Campaign Management Training (90 minutes)**

**Training Agenda:**
```
1. Campaign Planning (20 min)
   - Campaign types and strategies
   - Scheduling best practices
   - Budget and resource allocation
   - Store selection and targeting

2. Campaign Execution (30 min)
   - Task assignment and tracking
   - Mobile app walkthrough
   - Photo and execution validation
   - Real-time monitoring
   - Communication tools

3. Collaboration (20 min)
   - Brand integration
   - Approval workflows
   - Commenting and feedback
   - File sharing and assets

4. Advanced Features (20 min)
   - Automation rules
   - Templates and cloning
   - Bulk operations
   - Custom fields
```

**Deliverables:**
- Campaign management playbook
- Best practice guide
- Template examples
- Mobile app user guide

**Day 10-11: Field Team Training (45 minutes x 2 sessions)**

**Mobile App Training for Installers/Merchandisers:**
```
1. Getting Started (10 min)
   - App download and login
   - Profile setup
   - Notification settings

2. Task Management (15 min)
   - Viewing assigned tasks
   - Accepting assignments
   - Schedule navigation
   - Route planning

3. Execution (15 min)
   - Photo capture and upload
   - Form completion
   - Compliance checks
   - Submitting for approval

4. Communication (5 min)
   - In-app messaging
   - Support access
   - Reporting issues
```

**Deliverables:**
- Mobile quick-start guide (PDF)
- Video tutorials (bite-sized)
- FAQ document
- Support contact information

**Day 12-14: Integration and Testing**

**Integration Setup (if applicable):**
- CRM integration (Salesforce, HubSpot)
- Inventory management systems
- POS data feeds
- Reporting/BI tools
- SSO configuration

**Test Campaign:**
- Select small pilot scope (5-10 stores)
- Execute end-to-end workflow
- Validate data flow
- Test mobile app functionality
- Review reporting

**Milestone Checkpoint:**
- Day 14 check-in call (45 min)
- Review test campaign results
- Address any issues or confusion
- Adjust workflows as needed
- Plan for first live campaign

**Success Criteria for Week 2:**
- All staff trained (80%+ attendance)
- Campaign workflows configured
- Field team comfortable with mobile app
- Test campaign completed successfully
- Integrations validated

### Week 3: Execution (Days 15-21)

**Objectives:**
- Launch first live campaign
- Real-time execution support
- Troubleshooting and optimization
- Data validation
- Reporting setup

**Day 15-16: Pre-Launch Preparation**

**Campaign Review:**
- Final campaign configuration review
- Store network confirmation
- Staff assignments verified
- Communication plan ready
- Success metrics defined

**Go-Live Readiness:**
- Checklist completion review
- Risk assessment
- Rollback plan (if needed)
- Support escalation path
- Monitoring plan

**Day 17-19: Live Campaign Support**

**Active Monitoring:**
- Daily check-ins during execution
- Real-time issue resolution
- Field team support
- Performance tracking
- Quick adjustments as needed

**CSM Availability:**
- Extended support hours during launch
- Slack/Teams direct channel
- Phone/video on-demand
- Screen sharing for troubleshooting

**Day 20-21: Data Validation and Reporting**

**Review Activities:**
- Execution data validation
- Photo and compliance review
- Report generation
- Analytics dashboard setup
- Export and sharing

**Milestone Checkpoint:**
- Day 21 post-execution review (60 min)
- Campaign results analysis
- Lessons learned
- Optimization opportunities
- Next campaign planning

**Success Criteria for Week 3:**
- First live campaign executed
- 80%+ execution completion rate
- Minimal critical issues
- Reports generated and shared
- Team confidence in platform

### Week 4: Optimization (Days 22-30)

**Objectives:**
- Review and optimize first campaign
- Advanced features enablement
- Self-sufficiency validation
- Success metrics review
- Graduation to adoption phase

**Day 22-24: Campaign Optimization**

**Analysis Session (60 min):**
- Campaign performance review
- ROI calculation
- Efficiency analysis
- Quality assessment
- Comparative benchmarking

**Optimization Planning:**
- Workflow improvements
- Automation opportunities
- Template refinement
- Store network optimization
- Training reinforcement

**Day 25-27: Advanced Features Training (60 min)**

**Training Topics:**
```
1. Automation (20 min)
   - Automated task assignment
   - Recurring campaigns
   - Triggered notifications
   - Workflow rules

2. Reporting and Analytics (20 min)
   - Custom dashboards
   - Advanced filters
   - Data exports
   - Scheduled reports
   - Integration with BI tools

3. Optimization Tools (20 min)
   - Performance benchmarking
   - Store scoring
   - Budget optimization
   - Resource allocation
```

**Day 28-30: Graduation and Transition**

**First Business Review (60 min):**
```
1. Onboarding Recap (10 min)
   - Journey overview
   - Milestones achieved
   - Challenges overcome

2. Usage and Adoption (15 min)
   - User engagement metrics
   - Feature adoption
   - Campaign activity
   - ROI summary

3. Success Validation (15 min)
   - Goals achievement review
   - KPI performance
   - Customer testimonial/feedback
   - Quick wins celebration

4. Next 60 Days (15 min)
   - Transition to adoption phase
   - Ongoing support structure
   - Next campaign plans
   - Advanced use cases
   - Expansion opportunities

5. Q&A and Commitment (5 min)
   - Address concerns
   - Renewal conversation (if annual)
   - Reference/advocacy opportunity
```

**Deliverables:**
- Onboarding success report
- ROI documentation
- Best practices guide customized to their use case
- Ongoing support plan
- Next quarter success plan

**Success Criteria for Week 4:**
- Onboarding checklist 100% complete
- Team self-sufficient
- Positive feedback and NPS score
- Committed to ongoing usage
- Expansion interest identified

## Brand Onboarding (Lightweight 14-Day Journey)

### Overview
Brands have a simpler onboarding focused on campaign submission, collaboration with PSPs, and reporting access.

### Day 1: Account Setup and Orientation

**Welcome Email:**
- Login credentials
- Platform overview video (10 min)
- Quick-start guide
- Schedule orientation call

**Orientation Call (30 min):**
- Platform navigation
- Campaign submission process
- PSP collaboration model
- Reporting and visibility
- Asset library setup

### Days 2-7: Campaign Creation and Collaboration

**Self-Guided Setup:**
- Company profile completion
- Brand asset library upload
- Template exploration
- First campaign brief creation

**Training Video Series (5 videos x 5 min each):**
1. Creating a campaign brief
2. Working with PSP partners
3. Asset management
4. Approval workflows
5. Reporting and insights

**Milestone Check-in (Day 7, 30 min):**
- Review first campaign brief
- Q&A session
- Address any confusion
- Plan first campaign submission

### Days 8-14: First Campaign Execution

**Day 8-10: Campaign Submission**
- Submit first campaign to PSP partner
- Collaboration workflow
- Communication and feedback

**Day 11-13: Monitoring and Approval**
- Track campaign progress
- Review execution photos
- Approve completed work
- Provide feedback

**Day 14: Review and Graduation**
- Campaign results review (30 min)
- Reporting walkthrough
- Best practices discussion
- Next steps and ongoing usage

### Brand Onboarding Checklist

**Setup (5 tasks):**
- [ ] Account created
- [ ] Profile completed
- [ ] Assets uploaded
- [ ] PSP partners connected
- [ ] Users invited

**Training (3 tasks):**
- [ ] Orientation call completed
- [ ] Video series watched
- [ ] Documentation reviewed

**Execution (4 tasks):**
- [ ] First campaign brief created
- [ ] Campaign submitted to PSP
- [ ] Execution monitored
- [ ] Results reviewed

**Total: 12 tasks**

## Store/Installer Onboarding (Mobile-First 7-Day Journey)

### Overview
Stores and installers need lightweight, mobile-optimized onboarding focused on task execution and photo submission.

### Day 1: Account Activation

**Welcome SMS/Email:**
- App download link (iOS/Android)
- Login credentials
- Quick-start video (3 min)

**App First Launch:**
- Guided tour (interactive, 2 min)
- Profile setup
- Notification permissions
- Location services enable

### Days 2-3: Mobile App Training

**In-App Tutorials:**
- Task list navigation
- Accepting assignments
- Photo capture best practices
- Form completion
- Submission process

**Practice Task:**
- Sample task assigned
- Walk through entire flow
- Submit for review
- Receive feedback

### Days 4-7: First Real Tasks

**Day 4-5: First Assigned Tasks**
- Accept real assignments
- Execute in stores
- Submit photos and data
- Communicate with PSP

**Day 6-7: Optimization**
- Review performance feedback
- Address any issues
- Best practice tips
- Ongoing support access

### Store/Installer Onboarding Checklist

**Setup (3 tasks):**
- [ ] App downloaded and logged in
- [ ] Profile completed
- [ ] Notifications enabled

**Training (2 tasks):**
- [ ] In-app tutorial completed
- [ ] Practice task submitted

**Execution (3 tasks):**
- [ ] First real task completed
- [ ] Photo quality approved
- [ ] Payment/compensation received

**Total: 8 tasks**

## Onboarding Support Structure

### Roles and Responsibilities

**Customer Success Manager (CSM):**
- Primary point of contact
- Kickoff and check-in calls
- Training coordination
- Progress tracking
- Issue escalation
- Success validation

**Onboarding Specialist:**
- Training session delivery
- Technical configuration support
- Best practice guidance
- Documentation creation
- Handoff to CSM (day 30)

**Support Team:**
- Ticket-based support
- Technical troubleshooting
- How-to questions
- Bug reports

**Implementation Engineer (Enterprise only):**
- Complex integrations
- Custom configurations
- API development
- Data migration

### Communication Cadence

**PSP Customers:**
- Day 1: Kickoff call (60 min)
- Day 7: Progress check-in (30 min)
- Day 14: Milestone review (45 min)
- Day 21: Post-execution review (60 min)
- Day 30: Graduation and FBR (60 min)
- Ad-hoc: Email, Slack, as needed

**Brand Customers:**
- Day 1: Orientation call (30 min)
- Day 7: Check-in (30 min)
- Day 14: Review and graduation (30 min)

**Store/Installer:**
- Day 1: Welcome message (automated)
- Day 3: Check-in message (automated)
- Day 7: Feedback request (automated)
- Ad-hoc: In-app support

## Success Criteria and Graduation

### PSP Success Criteria

**Must-Have (Required for Graduation):**
- 100% onboarding checklist complete
- All training sessions attended (or recordings watched)
- First campaign successfully executed
- 80%+ team adoption (invited users active)
- Reports generated and reviewed
- Customer satisfaction score 4+/5

**Nice-to-Have (Bonus):**
- Advanced features in use
- Integration operational
- Second campaign planned
- Positive testimonial/feedback
- Referral opportunity identified

### Brand Success Criteria

**Must-Have:**
- 100% onboarding checklist complete
- First campaign submitted and executed
- Reporting reviewed
- Customer satisfaction 4+/5

### Store/Installer Success Criteria

**Must-Have:**
- App installed and profile complete
- First task completed successfully
- Photo quality approved
- App rated 4+/5 stars

### Graduation Process

**Day 30 Review:**
- Validate all success criteria met
- Document ROI and value achieved
- Gather testimonial/feedback
- Transition to adoption phase
- Handoff from onboarding specialist to CSM (if different)

**Post-Onboarding:**
- Move to standard CSM cadence (monthly check-ins)
- Ongoing training opportunities
- Community access
- Advanced features enablement
- Expansion conversations

## Handoff to Customer Success

### Onboarding-to-CS Transition

**Day 25-28: Preparation**
- CSM reviews onboarding progress
- Notes from onboarding specialist
- Customer goals and success criteria
- Relationship context and stakeholders
- Identified expansion opportunities

**Day 30: Formal Handoff**
- Onboarding specialist introduces CSM (if new)
- Review ongoing support structure
- Set expectations for next phase
- Schedule next check-in (30 days out)

**Post-Handoff Documentation:**
- Onboarding summary report
- Customer profile update
- Health score baseline
- Success plan for next 90 days

## Onboarding Metrics and KPIs

### Leading Indicators

**Engagement Metrics:**
- Checklist completion rate (target: 100%)
- Training attendance rate (target: 90%)
- Time to first login (target: <24 hours)
- Time to first campaign creation (target: <7 days)
- Team activation rate (target: 80%)

**Usage Metrics:**
- Login frequency (target: 3x/week)
- Feature adoption (target: 50% of core features)
- Campaign creation (target: 1+ in first 30 days)
- Mobile app downloads (target: 100% of field teams)

### Lagging Indicators

**Success Metrics:**
- Onboarding completion rate (target: 95%)
- Time to first value (target: <14 days)
- First campaign success rate (target: 90%)
- Onboarding CSAT (target: 4.5+/5)
- Onboarding NPS (target: 50+)

**Business Metrics:**
- Activation rate (target: 90% of new customers)
- Early churn rate (target: <5% in first 90 days)
- Time to renewal conversation (tracked)
- Upsell opportunity identification (target: 30%)

### Reporting and Optimization

**Weekly Onboarding Report:**
- New customers in onboarding
- Progress by customer (checklist % complete)
- At-risk customers (red flags)
- Upcoming milestones and graduations
- Support ticket volume from onboarding customers

**Monthly Onboarding Review:**
- Cohort analysis (completion rates, time to value)
- Training effectiveness (attendance, satisfaction)
- Resource utilization (most-viewed docs, videos)
- Bottlenecks and drop-off points
- Customer feedback themes

**Quarterly Optimization:**
- Review and update onboarding playbook
- Refine success criteria
- Improve training materials
- Optimize communication cadence
- A/B test onboarding variations
